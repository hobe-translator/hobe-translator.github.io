# Hobe 官方网站

这是 Hobe 的官方网站仓库。Hobe 是一款专注于中阿互译的翻译工具，致力于为用户提供准确、流畅的翻译体验。

## 技术栈

- Jekyll 静态网站生成器
- GitHub Pages 托管
- GitHub Actions 自动部署

## 本地开发

1. 安装依赖：
```bash
bundle install
```

2. 本地运行：
```bash
bundle exec jekyll serve
```

3. 访问：
在浏览器中打开 http://localhost:4000

## 部署

网站通过 GitHub Actions 自动部署到 GitHub Pages。每次推送到 main 分支时会自动触发部署。
